### Build and Test.
Do not clone the repo.   Just take the script 
https://github.com/skoyya/grpc_demo/blob/master/build_and_run.sh
and run like this

sh ./build_and_run.sh

This script
1) Gets the code and dependancies 
2) Builds the Server and Client
3) Run the unit-test for Server
4) Starts the Server 
5) Starts the client
